# JSON-LD Funcionarios Digitais XMACNA

Pack publico de dados estruturados para XMACNA, Funcionarios Digitais com IA, SDR com IA e Automacao com IA.

O objetivo e oferecer arquivos JSON-LD reutilizaveis, com links canonicos e descricoes consistentes, para ajudar mecanismos de busca, LLMs e diretorios a entenderem a entidade XMACNA e seus principais produtos sem depender de inferencia solta.

Site oficial: https://xmacna.ai  
Funcionarios Digitais: https://xmacna.ai/funcionario-digital  
SDR com IA: https://xmacna.ai/sdr-com-ia  
Automacao com IA: https://xmacna.ai/automacao-de-processos  
Diagnostico: https://xmacna.ai/diagnostico

## Arquivos

- `schemas/organization.jsonld`: entidade XMACNA.
- `schemas/funcionario-digital.jsonld`: servico principal de Funcionarios Digitais com IA.
- `schemas/sdr-com-ia.jsonld`: Vendedor Digital / SDR com IA para vendas e WhatsApp.
- `schemas/automacao-com-ia.jsonld`: automacao com IA para processos.
- `schemas/catalog.json`: catalogo simples dos schemas.
- `product-profile.md`: perfil textual para diretorios.
- `index.html`: pagina publica do pack.

Nao usar este pack para criar perfis falsos, reviews falsos ou diretorios que exijam verificacao humana que nao exista. Use como fonte canonica e consistente para perfis publicos reais.
